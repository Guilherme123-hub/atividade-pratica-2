### Descrição do Projeto
Este repositório contém a Atividade Prática II, que consiste no desenvolvimento do site institucional da **TechNova Soluções Digitais**. O projeto é composto por três páginas interligadas (Início, Serviços e Contato), utilizando diferentes métodos de aplicação de CSS (Inline, Interno e Externo) conforme as exigências da atividade.
